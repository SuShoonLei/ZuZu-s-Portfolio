# Myat Myint Zu — Portfolio

Personal portfolio site for Myat Myint Zu, HR Leader / Human Resources Specialist (Bangkok).

## Stack

- React + Vite (JavaScript)
- React Three Fiber + three.js (`Antigravity` cursor particles)
- WebGL metallic paint shader (`MetallicPaint` name treatment)
- Plain CSS with design tokens

## Develop

```bash
cd portfolio
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Static output is written to `dist/` and can be deployed to any static host.
